安装指南

简单的安装步骤，快速开始使用

Windows 安装

  1. 下载 Windows 安装包 (.exe)
  2. 双击运行安装程序
  3. 按向导完成安装
  4. 启动应用并配置
查看详细教程

macOS 安装

  1. 下载 .dmg 安装包
  2. 打开并挂载镜像
  3. 拖拽到 应用程序 文件夹
  4. 首次运行通过 Gatekeeper
查看详细教程

Linux 安装

  1. 获取 AppImage / deb / rpm 包
  2. 为 AppImage 添加执行权限
  3. 或使用包管理器安装
  4. 运行并创建配置目录
